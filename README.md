# Zelda Quiz Game

This is a quiz game where you guess the names of characters from *The Legend of Zelda: Breath of the Wild*.

---

## About the Project

- Built as a school/portfolio project.
- Frontend: HTML, CSS, JavaScript.
- Gameplay: Display a character's image and let the player guess the name.
- Results are shown with the number of correct and incorrect answers, as well as a comparison to previous attempts.
